Hub Allocation requires data not in the current CSV
The current CSV contains buffer math (ADU / TOR / TOG / NFP / OH / OO / QD) — enough for Buffer Detail and Order Recommendation. Hub Allocation needs supplier-level + hub-level data that isn't present.
To enable Hub Allocation, add a suppliers.csv with these columns:
MSKU_CODE,SUPPLIER_CODE,SUPPLIER_NAME,UNIT,PROCUREMENT_TYPE,HUB_STOCK_CLEARED_PCT,LAST_PO_QTY,LAST_PO_CLEARED_QTY,RECOMMENDED
And optionally a colour_size_matrix.csv for the colour × size + design split steps:
MSKU_CODE,COLOUR,SIZE,MRP_BAND,DESIGN_CODE,WEIGHT_PCT

Drop these in Stock Upload (a future endpoint) and this tab will light up with the supplier wizard from the mockup.
